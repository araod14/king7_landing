# KING7 BOXCROSS Landing Page

A mobile-first, high-performance landing page for a CrossFit Box in Barcelona, Venezuela.

## Tech Stack

- **Framework:** Next.js 15+ (App Router)
- **Styling:** Tailwind CSS 4.0
- **Language:** TypeScript
- **Deployment:** Optimized for Vercel

## Features

- 📱 **Mobile-First Design:** Optimized for small screens with a sticky CTA.
- 🎨 **Energetic Aesthetic:** Bold black/yellow/white palette for a strong fitness look.
- ⚡ **Performance:** Fast loading with Next.js App Router and optimized CSS.
- 💬 **Direct Contact:** Floating WhatsApp button for quick inquiries.
- 🧩 **Component-Based:** Clean, reusable React components.
- 📜 **Smooth Navigation:** In-page anchor links for a seamless user experience.

## Project Structure

- `src/app`: Application routes and global styles.
- `src/components`: UI building blocks (Navbar, Hero, Benefits, Coach, Gallery, Pricing, etc.).
- `public`: Static assets (Logos/Images placeholders).

## Development

```bash
npm install
npm run dev
```

## Deployment

Simply push to a GitHub repository and connect it to Vercel.
